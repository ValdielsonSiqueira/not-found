# @FIAP/not-found

## Descrição

Microfrontend simples exibido quando uma rota não mapeada é acessada (Página 404).

## Tecnologias

- **Framework**: React.
- **Linguagem**: TypeScript.
- **Arquitetura**: Single-SPA.
- **Design System**: [`@valoro/ui`](https://www.npmjs.com/package/@valoro/ui).

## Pré-requisitos

- **Node.js**: Versão LTS.
- **Gerenciador de Pacotes**: pnpm.

## Como Rodar

1. Instale as dependências:

   ```bash
   pnpm install
   ```

2. Inicie o servidor:
   ```bash
   pnpm start
   ```

## Funcionalidades

- Exibição amigável de erro 404.
- Link para retorno à Home.
